# PBEL project 
Project for pbel 
Ai powered health diagnosis 
Modules used - 
panda
numoy
joblib
scikit learn
joblib
seaborn
streamlit
matplotlib
This project consist of 4 files
1. app.py - basically the ui for this project uses streamlit
2. model.py - basically whole model of the project what methods and algorithms were used
3. gsd.py - generate synthetic data, basically used to generate synthetic datasets for testing or machine learning experiments.
4. zip file - Dataset file in zip as it was large in size.
To run it-
1. extract the zip dataset so that model can use the dataset.
2. open terminal and run python model.py
it will use the dataset and train the model.
3. now to check the model run streamlit run app.py and it will provide a local link on which you can enter the data and check the prediction.
